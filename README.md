# AllReacts for Facebook Stories

Tăng cường trải nghiệm xem Story trên Facebook với hàng ngàn cảm xúc khác nhau và tính năng ẩn trạng thái "đã xem".

## 🚀 Tính năng chính
- **Kho Emoji khổng lồ**: Cung cấp hơn 1,800+ emoji từ chuẩn mới nhất.
- **Ẩn trạng thái "Đã xem"**: Xem story của người khác mà không để lại dấu vết.
- **Tìm kiếm thông minh**: Dễ dàng tìm thấy emoji yêu thích theo tên hoặc biểu tượng.
- **Giao diện mượt mà**: Menu emoji tích hợp ngay trong giao diện xem Story của Facebook.

## 🆕 Cập nhật phiên bản 2.1.0
- **Nâng cấp Emoji 16.0**: Cập nhật toàn bộ thư viện emoji lên chuẩn Emoji 16.0 mới nhất (thông qua `emoji-datasource-facebook`).
- **Thêm Emoji mới**: Bổ sung các biểu tượng hot như **HEART HANDS (🫶)** và nhiều cảm xúc mới khác.
- **Tối ưu hóa tốc độ tải**: Thay đổi logic ưu tiên tải dữ liệu từ local để đảm bảo các cập nhật mới nhất luôn được hiển thị ngay lập tức.
- **Đồng bộ hóa CDN**: Sử dụng link CDN `16.0.0` để đảm bảo hình ảnh sắc nét và tải nhanh.

## 🛠️ Cài đặt
1. Tải về hoặc clone repository này.
2. Mở trình duyệt Chrome/Edge và truy cập `chrome://extensions/`.
3. Bật **Developer mode** (Chế độ nhà phát triển).
4. Chọn **Load unpacked** (Tải tiện ích đã giải nén) và chọn thư mục chứa project này.
